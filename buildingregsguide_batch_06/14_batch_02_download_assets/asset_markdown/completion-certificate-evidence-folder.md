# Completion Certificate Evidence Folder

Type: document organiser

## Intended use

This asset should be a simple PDF/printable that helps a homeowner prepare evidence before contacting building control, a designer, a builder or a competent installer.

## Sections

- Application details
- Inspection notes
- Installer certificates
- Photos
- Calculations
- Product specs
- Final certificate

## Conversion note for Codex

Render as an HTML download landing page first. PDF generation can be added later. Keep fields short and printable; avoid long prose inside the asset itself.
