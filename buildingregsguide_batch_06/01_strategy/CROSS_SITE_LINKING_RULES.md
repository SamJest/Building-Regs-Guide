# Cross-Site Linking Rules

Use contextual links only.

## From UKPlanningGuide to BuildingRegsGuide

Link only where a page naturally reaches the building-control question:

- planning permission vs building regulations FAQ
- extension pages after planning route answer
- loft/garage/outbuilding pages where building control is a separate next step
- tool results that say planning is likely settled but building regulations remain

## From BuildingRegsGuide back to UKPlanningGuide

Link only where the user may still need planning checks:

- visible external changes
- extensions/lofts/outbuildings/porches
- conservation areas/listed buildings
- heat pumps/solar panels/planning-sensitive external equipment
- driveways/dropped kerbs

## Avoid

- sitewide footer cross-links
- repetitive exact-match anchors
- duplicate articles across both domains
- pages whose only purpose is to funnel traffic to the other site

## Anchor style

Use descriptive natural anchors such as:

- `check the planning route first`
- `planning permission may still matter`
- `separate building regulations checklist`
- `building control evidence to keep`
