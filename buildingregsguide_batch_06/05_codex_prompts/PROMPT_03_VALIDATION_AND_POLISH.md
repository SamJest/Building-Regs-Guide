# Prompt 03 — Validation and polish

Adapt validation from the UKPlanningGuide scaffold, but remove planning-specific checks.

Validation must check:

- duplicate slugs/paths
- missing title/meta description
- missing canonical
- missing official source block
- missing route registry entry
- missing sitemap inclusion
- broken internal links
- England wording leakage onto Wales/Scotland pages
- tool pages missing result root or JS
- download pages missing print/copy controls
- pages making compliance guarantees

Then rebuild, run validation, and provide the final route count and any skipped checks.
