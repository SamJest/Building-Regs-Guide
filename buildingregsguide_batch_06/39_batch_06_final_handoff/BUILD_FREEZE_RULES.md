# Build freeze rules for Codex

Do not add these before first launch:

- accounts or login
- payment flows
- display ads
- affiliate blocks
- lead-generation forms pretending to give advice
- AI chatbot answers on compliance
- scraped local authority pages
- unverified Scotland/Wales/NI copies
- hundreds of indexable programmatic pages

Allowed for first launch:

- static pages
- tools with conservative logic
- source/version panels
- printable downloads
- no-login local dashboard
- internal search
- sitemap/robots/schema
- restrained cross-linking with UKPlanningGuide
