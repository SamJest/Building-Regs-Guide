# Acceptance criteria

The handover is acceptable when:

1. The site builds without errors.
2. The homepage, core hubs, phase 1 project pages, tools and downloads are navigable.
3. Every regulatory page has a source/version block.
4. Tool results never claim to grant approval.
5. Higher-risk building inputs route to BSR/specialist guidance.
6. Approved Document L/F pages are version-aware.
7. Sitemap includes only indexable pages.
8. Candidate pages that are not complete remain draft/noindex.
9. Internal search returns relevant pages/tools/downloads.
10. Print views are readable and do not waste ink with navigation clutter.
11. The dashboard stores data locally and clearly says so.
12. Cross-site links to UKPlanningGuide are helpful, not spammy.
13. The final build report lists published, draft/noindex and blocked routes.
