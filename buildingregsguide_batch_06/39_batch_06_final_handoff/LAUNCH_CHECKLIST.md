# Launch checklist

## Before build

- Confirm domain choice and canonical URL.
- Confirm UKPlanningGuide cross-link targets.
- Confirm source snapshot is still current.
- Confirm no pages imply formal approval.

## Build checks

- Site builds cleanly.
- No duplicate slugs.
- No broken internal links.
- Source panels present.
- Tool result pages print cleanly.
- Dashboard localStorage can be cleared.
- Search index generated.
- Sitemap excludes noindex/draft pages.

## Content checks

- Planning topics route to UKPlanningGuide for depth.
- Building regulations topics stay on BuildingRegsGuide.
- England-first wording is clear.
- HRB cases route to BSR/specialist advice.
- Approved Document pages show version/date warnings.

## Production checks

- Submit sitemap in Search Console.
- Check robots.txt.
- Check canonical tags.
- Check mobile rendering.
- Check analytics events without collecting unnecessary personal data.
- Record first crawl/indexing baseline.
