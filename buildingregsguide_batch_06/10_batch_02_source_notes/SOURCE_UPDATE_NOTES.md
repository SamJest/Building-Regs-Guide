# Batch 2 source update notes

Checked on 2026-06-04. Treat GOV.UK as the primary source for rule-sensitive content.

Important current points for Codex:

1. Building regulations approval is separate from planning permission. Pages and tools must never imply that one replaces the other.
2. A competent person scheme can mean the homeowner does not need to apply separately, but the tool must still explain certificates and sale/resale evidence.
3. Higher-risk building pages must route users to Building Safety Regulator guidance and must not be simplified into the normal homeowner route.
4. Approved Document L has a 2026 publication. Part L pages and energy-efficiency tools need explicit version checks and transition wording.
5. The site should explain that building control is regulatory oversight, not a workmanship guarantee or design service.
