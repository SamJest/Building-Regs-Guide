# Download internal linking map

## Rules

Every project page should link to:

1. the most relevant project-specific download,
2. the building-control route checker,
3. the Approved Document router,
4. the inspection-stage checklist generator,
5. UKPlanningGuide where planning permission or lawful development may be relevant.

Every download page should link back to:

1. its project page,
2. the relevant tool,
3. the evidence folder / project dashboard,
4. official source page,
5. at least two related downloads.

## Priority sitewide CTAs

- Start with the building regs route checker.
- Save your result to a no-login project dashboard.
- Print the before-you-start checklist.
- Build your inspection timeline.
- Track certificates before completion.
- Check planning permission separately on UKPlanningGuide.

## Prevent bloat

Do not add every asset everywhere. Use project tags and tool flags. A page should show 1 primary download and 2 secondary downloads only.
