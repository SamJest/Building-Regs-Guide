# Approved Document hub build notes

Batch 3 turns the seed Approved Document pages into implementation-ready hub drafts.

Codex should build these as a consistent `/approved-documents/{part}/` family with:

- Official source/version block near the top.
- Plain-English homeowner scope.
- Common projects where the Part appears.
- Evidence checklist.
- Red flags.
- Related Approved Documents.
- Tool entry points.
- UKPlanningGuide cross-link only where planning permission overlap is real.

Do not reproduce the official Approved Documents. Summarise route-finding and evidence requirements, then link out to GOV.UK.
