# Approved Document M: Access to and use of buildings

Approved Document M is the practical guidance route for **accessibility and use, including accessible entrances, circulation, sanitary provision and visitability**. This page should help a homeowner understand when this Part is likely to matter, what evidence to keep, and which project pages to read next.

This is not a substitute for a designer, structural engineer, registered installer, building control body or the official Approved Document. Codex should render this as a plain-English hub with prominent links to the official source and to project-specific checks.

## When this Part commonly appears

- new dwellings
- extensions affecting entrances
- ground-floor WC changes
- level thresholds
- flats/common parts

## What the homeowner normally needs to prove

- access layout
- threshold details
- door/corridor widths where relevant
- WC layout
- ramp/step details

## Common red flags

- removing only ground-floor WC
- narrowing circulation routes
- steps at new entrances
- work to communal residential access

## Related Approved Documents

This Part often overlaps with Part K, Part G, Part B. The page must not present any Part in isolation where a normal project usually crosses several requirements.
