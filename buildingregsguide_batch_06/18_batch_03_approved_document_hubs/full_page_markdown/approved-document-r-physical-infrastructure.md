# Approved Document R: Physical infrastructure for electronic communications

Approved Document R is the practical guidance route for **in-building physical infrastructure for high-speed electronic communications networks**. This page should help a homeowner understand when this Part is likely to matter, what evidence to keep, and which project pages to read next.

This is not a substitute for a designer, structural engineer, registered installer, building control body or the official Approved Document. Codex should render this as a plain-English hub with prominent links to the official source and to project-specific checks.

## When this Part commonly appears

- new dwellings
- major residential new-build projects
- some conversion/new unit projects

## What the homeowner normally needs to prove

- ducting/infrastructure specification
- network termination point location
- developer/installer notes

## Common red flags

- creating new dwellings
- omitting telecoms infrastructure until late build
- routes blocked by structure/insulation

## Related Approved Documents

This Part often overlaps with Part P, Part L. The page must not present any Part in isolation where a normal project usually crosses several requirements.
