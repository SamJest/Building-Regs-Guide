# Approved Document C: Site preparation, contaminants and moisture

Approved Document C is the practical guidance route for **site preparation, moisture resistance, radon/contaminants and damp protection**. This page should help a homeowner understand when this Part is likely to matter, what evidence to keep, and which project pages to read next.

This is not a substitute for a designer, structural engineer, registered installer, building control body or the official Approved Document. Codex should render this as a plain-English hub with prominent links to the official source and to project-specific checks.

## When this Part commonly appears

- extensions
- basements
- garage conversions
- solid floor upgrades
- work on damp-prone walls
- new floors over ground

## What the homeowner normally needs to prove

- damp-proof course/damp-proof membrane details
- ground condition notes
- radon/contaminant checks where relevant
- floor build-up specification
- photos before covering membranes

## Common red flags

- below-ground rooms
- historic damp
- made ground/contaminated land possibility
- floor levels close to external ground
- bridged damp-proof courses
