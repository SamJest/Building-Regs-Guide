# Approved Document T: Toilet accommodation

Approved Document T is the practical guidance route for **toilet accommodation requirements introduced as a separate Approved Document for relevant buildings**. This page should help a homeowner understand when this Part is likely to matter, what evidence to keep, and which project pages to read next.

This is not a substitute for a designer, structural engineer, registered installer, building control body or the official Approved Document. Codex should render this as a plain-English hub with prominent links to the official source and to project-specific checks.

## When this Part commonly appears

- mostly non-domestic projects
- mixed-use projects
- some publicly accessible building work

## What the homeowner normally needs to prove

- toilet layout
- accessibility notes
- separate accommodation schedule where relevant

## Common red flags

- publicly accessible premises
- conversion to commercial/mixed-use
- assuming domestic-only rules apply to non-domestic use

## Related Approved Documents

This Part often overlaps with Part G, Part M. The page must not present any Part in isolation where a normal project usually crosses several requirements.
