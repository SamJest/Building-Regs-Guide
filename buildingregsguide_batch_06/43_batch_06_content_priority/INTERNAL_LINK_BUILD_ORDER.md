# Internal link build order

1. Homepage to start-here tools, project hubs and planning-vs-building-regs page.
2. Project hubs to relevant Approved Document hubs.
3. Project hubs to relevant tools.
4. Tool results to downloads and project pages.
5. Downloads back to their source project/tool pages.
6. Approved Document hubs to practical project examples.
7. Planning-overlap pages to UKPlanningGuide.
8. UKPlanningGuide inbound links back to BuildingRegsGuide only where the user needs building control detail.

Link limits:

- Keep contextual body links focused.
- Avoid sitewide blocks with dozens of links.
- Prefer 3-6 related links per page section.
- Avoid linking every mention of every term.
