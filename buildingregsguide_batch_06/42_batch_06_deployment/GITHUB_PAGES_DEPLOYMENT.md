# GitHub Pages deployment notes

Use if the final output is static.

1. Commit generated site and source.
2. Configure build command if using a generator.
3. Output to `/dist` or equivalent.
4. Add custom domain after DNS is ready.
5. Add `CNAME` if required by GitHub Pages.
6. Confirm sitemap, robots and canonical URLs use production domain.
7. Test 404 page and static asset paths.
