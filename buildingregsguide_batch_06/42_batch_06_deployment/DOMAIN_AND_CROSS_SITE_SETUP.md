# Domain and cross-site setup

Assumed domain: `buildingregsguide.co.uk`.

Cross-link rules:

- UKPlanningGuide -> BuildingRegsGuide where the user needs building control, Approved Documents, inspections or completion evidence.
- BuildingRegsGuide -> UKPlanningGuide where the user needs planning permission, permitted development, prior approval, lawful development certificates or planning enforcement.

Keep the two sites distinct. This is a topical ecosystem, not a duplicated site network.
