# Deployment options

Recommended first launch: static hosting.

## Best fit

Cloudflare Pages or Netlify are easiest for a static generated site with redirects, preview deploys and simple build logs.

## Also viable

GitHub Pages if the output is simple static HTML/CSS/JS and the build process is stable.

## Avoid for first launch

Do not start with a full CMS, user accounts or database unless a future feature genuinely needs it. The no-login dashboard should stay local-only at launch.
