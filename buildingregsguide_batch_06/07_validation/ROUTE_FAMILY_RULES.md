# Route Family Rules

## Guide pages

Must include: direct answer, route card, source block, related tool, related download.

## Project pages

Must include: approval trigger summary, project-specific checks, evidence checklist, planning handoff where relevant.

## Approved Document pages

Must include: what the document covers, common projects, version-aware source block, related guides.

## Tool pages

Must include: static form, JS result engine, official source links in result, print/save/share controls if inherited from base.

## Download pages

Must be crawlable HTML pages. PDF generation is optional later.

## Jurisdiction pages

Must prevent wrong-country assumptions. Wales and Scotland pages are guardrails in Batch 1, not full local engines.
