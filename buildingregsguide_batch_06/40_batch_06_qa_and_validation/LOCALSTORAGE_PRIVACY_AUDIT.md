# LocalStorage privacy audit

The no-login dashboard should store only practical project checklist data on the user's own device.

Do store:

- project type
- saved tool result summary
- checklist completion state
- evidence item labels
- user-entered notes if clearly local-only

Do not store:

- precise home address by default
- full names of contractors unless user chooses to type them
- sensitive personal data
- payment details
- anything sent to a server unless a backend and privacy policy are deliberately implemented later

Required UI copy:

`This dashboard is saved on this device only. It is not an official record and does not replace building control, installer certificates or professional advice.`
