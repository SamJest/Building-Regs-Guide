# Prompt 11: implement internal link graph

Use `internal_link_graph_v5.json` and `relatedContentEngine.js` to render related content blocks.

Requirements:

- Project pages show related Approved Documents, tools and downloads.
- Approved Document pages show related project pages.
- Question pages link to parent project, tool and download.
- Evidence pages link to completion certificate/download/dashboard routes.
- Add an orphan detection report.
- Do not render irrelevant links just because a keyword matches.
