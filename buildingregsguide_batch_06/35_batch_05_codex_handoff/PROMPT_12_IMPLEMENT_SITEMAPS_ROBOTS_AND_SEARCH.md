# Prompt 12: implement sitemap, robots and search index

Use `sitemapGenerator.js`, sitemap contract and search index files.

Requirements:

- Generate `/sitemap.xml` from passing routes only.
- Generate `/robots.txt` with sitemap reference.
- Exclude tool result states, print views, API routes and draft/noindex pages.
- Generate `/search.json` using the search index contract.
- Add a basic search page that reads the local search index.
