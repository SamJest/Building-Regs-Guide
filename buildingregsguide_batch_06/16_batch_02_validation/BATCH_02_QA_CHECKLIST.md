# Batch 2 QA checklist

## Content

- [ ] Every Batch 2 page has a clear plain-English answer within the first 150 words.
- [ ] Every page distinguishes building regulations from planning permission where relevant.
- [ ] Every project page has evidence, inspection and red-flag sections.
- [ ] Higher-risk pages route to BSR and avoid normal homeowner simplification.
- [ ] Competent-person pages explain certificate retention.

## Tools

- [ ] No result says the user is approved or legally safe to start.
- [ ] Already-started work routes to regularisation/evidence recovery.
- [ ] Higher-risk flag overrides normal domestic routing.
- [ ] Competent-person results still warn about separate planning/listed/freeholder issues.
- [ ] Tool results are printable and source-linked.

## SEO/internal linking

- [ ] Every core explainer links to at least two project pages.
- [ ] Every project page links to Do I Need Approval, Full Plans vs Building Notice, Completion Certificate and at least one relevant tool.
- [ ] Download assets link from relevant project/tool results.
