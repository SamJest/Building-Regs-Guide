# Future Site Reuse Notes

## Best Sister Site Candidate

BuildingRegsGuide is the strongest next site because it naturally follows UKPlanningGuide:

- UKPlanningGuide answers: do I need planning permission?
- BuildingRegsGuide answers: do I need building regulations approval and what evidence/certificates matter?

## Reuse Strategy

Reuse the architecture, not the planning content.

Keep:

- static generator
- page-family structure
- source-backed blocks
- tool pages
- download assets
- sitemap and validation discipline
- answer-first layout

Replace:

- planning rule datasets
- planning permission tools
- planning-specific SEO copy
- local planning authority assumptions
- project route logic that depends on permitted development

## First BuildingRegsGuide Scope

Recommended first build:

- homepage
- building regulations basics hub
- extensions
- loft conversions
- garage conversions
- outbuildings/garden rooms
- drainage and waste
- structural alterations
- windows and doors
- electrical work
- heating and boilers
- completion certificates
- building notice vs full plans
- competent person schemes
- 5-8 tools
- 10-20 printable checklists

## Cross-Site Linking Principles

Use contextual links only:

- From UKPlanningGuide to BuildingRegsGuide where planning and building regs split.
- From BuildingRegsGuide back to UKPlanningGuide where planning route must be checked first.

Avoid:

- sitewide footer cross-links
- exact-match repetitive anchors
- duplicate articles across domains
- doorway pages created only to funnel traffic
