Building regulations downloads

Printable checklists and evidence sheets for homeowner project files.

How to use this section

Start with the page closest to your project, then use the linked tool or printable checklist to turn the guidance into questions for your building control body, designer, builder or registered installer.

Every published page keeps planning permission separate from building regulations approval and links back to official sources so you can re-check the current version before relying on it.

For a cleaner project file, save the likely route, record who is responsible for each certificate, and keep dated evidence before important work is covered up. If the work includes structure, fire safety, drainage, ventilation, electrics, heating or missing historic paperwork, confirm the route before committing money.

Source and version panel

This page is England-first unless a nation is named. Re-check the linked source before starting work, appointing trades or submitting an application.

  • GOV.UK Building regulations approval Checked 2026-06-05
    • Building regulations approval is different from planning permission and a project may need both.
    • Some work does not require the homeowner to apply directly if a competent person scheme member is used.
    • England-first guidance should be labelled and not implied to cover Scotland, Wales or Northern Ireland without separate checks.

BuildingRegsGuide is an independent, source-linked guide for homeowners. It does not grant building regulations approval.
